Introducción

Python es un lenguaje de programación interpretado y de alto nivel. Es conocido por su sintaxis clara y legible que favorece el desarrollo rápido de aplicaciones.

Variables

En Python, una variable es un contenedor para almacenar valores. Puedes asignar un valor a una variable utilizando el operador de asignación `=`.

Tipos de Datos

Python es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar explícitamente el tipo de una variable. Algunos tipos de datos comunes en Python son:

  • Enteros (int)
  • Flotantes (float)
  • Cadenas de caracteres (str)
  • Listas (list)
  • Tuplas (tuple)
  • Diccionarios (dict)
  • Booleanos (bool)

Ejemplos

Aquí hay algunos ejemplos de variables y tipos de datos en Python:

edad = 25
nombre = "Juan"
precio = 9.99
lista = [1, 2, 3, 4, 5]
tupla = (10, 20, 30)
diccionario = {"clave": "valor"}
