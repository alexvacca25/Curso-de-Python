Ejemplos de Variables en Python

Python es un lenguaje de programación de alto nivel que es conocido por su sintaxis clara y legible. Aquí hay algunos ejemplos de cómo se pueden utilizar variables en Python:

Ejemplos

  • Ejemplo 1: Variables Numéricas

    En Python, puedes utilizar variables para almacenar números enteros o decimales.

    edad = 25
precio = 9.99

  • Ejemplo 2: Variables de Cadena

    Las variables de cadena se utilizan para almacenar texto en Python.

    nombre = "Juan"
mensaje = 'Hola, ¿cómo estás?'

  • Ejemplo 3: Variables Booleanas

    Las variables booleanas pueden ser verdaderas o falsas.

    activo = True
entregado = False
